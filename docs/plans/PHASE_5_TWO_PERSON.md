# Phase 5: Two-Person Connection & Conflict Resolution

## Vision

Enable two people to use Feelings & Needs together to build mutual understanding, resolve conflicts, and strengthen their connection through structured NVC dialogue.

This is the culmination of the platform's promise: not just individual practice, but **real communication transformation between people**.

---

## Why This Phase Matters

### The Gap in Current Solutions

Individual NVC practice is valuable, but the real challenge is applying it *with another person*:
- Knowing your own feelings/needs doesn't guarantee being heard
- The other person may not know NVC
- Real-time conversation is harder than practice
- Historical hurts create emotional charge
- Power dynamics complicate dialogue

### What Phase 5 Enables

1. **Preparation Together**: Both parties do individual work before connecting
2. **Empathy Exchange**: See how well you understand each other
3. **Shared Needs Discovery**: Find common ground
4. **Strategy Co-Creation**: Brainstorm solutions together
5. **Agreement Tracking**: Commit to actions and follow up

### Database Schema

```sql
-- Shared sessions
CREATE TABLE shared_sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  topic TEXT NOT NULL,
  relationship_type VARCHAR(50),
  status VARCHAR(50) DEFAULT 'pending',
  created_at TIMESTAMP DEFAULT NOW(),
  expires_at TIMESTAMP DEFAULT NOW() + INTERVAL '7 days',
  completed_at TIMESTAMP
);

-- Session participants
CREATE TABLE session_participants (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  session_id UUID REFERENCES shared_sessions(id) ON DELETE CASCADE,
  role VARCHAR(20) NOT NULL, -- 'initiator' or 'invitee'
  user_id UUID REFERENCES users(id),
  device_id VARCHAR(255),
  name VARCHAR(100),
  joined_at TIMESTAMP,

  -- OFNR Preparation
  observation TEXT,
  feelings TEXT[],
  needs TEXT[],
  request TEXT,
  prep_completed_at TIMESTAMP,

  -- Empathy guesses about the other
  guessed_feelings TEXT[],
  guessed_needs TEXT[],
  guessed_context TEXT,
  empathy_completed_at TIMESTAMP
);

-- Strategies brainstormed
CREATE TABLE session_strategies (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  session_id UUID REFERENCES shared_sessions(id) ON DELETE CASCADE,
  suggested_by UUID REFERENCES session_participants(id),
  content TEXT NOT NULL,
  meets_needs TEXT[],
  created_at TIMESTAMP DEFAULT NOW(),
  ai_suggested BOOLEAN DEFAULT FALSE
);

-- Strategy reactions
CREATE TABLE strategy_reactions (
  strategy_id UUID REFERENCES session_strategies(id) ON DELETE CASCADE,
  participant_id UUID REFERENCES session_participants(id) ON DELETE CASCADE,
  reaction VARCHAR(20), -- 'like', 'discuss', 'disagree'
  PRIMARY KEY (strategy_id, participant_id)
);

-- Agreements
CREATE TABLE agreements (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  session_id UUID REFERENCES shared_sessions(id) ON DELETE CASCADE,
  created_at TIMESTAMP DEFAULT NOW(),
  check_in_date DATE,
  status VARCHAR(50) DEFAULT 'active'
);

-- Agreement items
CREATE TABLE agreement_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  agreement_id UUID REFERENCES agreements(id) ON DELETE CASCADE,
  content TEXT NOT NULL,
  alex_agreed BOOLEAN DEFAULT FALSE,
  jordan_agreed BOOLEAN DEFAULT FALSE,
  sort_order INT
);

-- Agreement check-ins
CREATE TABLE agreement_checkins (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  agreement_id UUID REFERENCES agreements(id) ON DELETE CASCADE,
  participant_id UUID REFERENCES session_participants(id),
  item_id UUID REFERENCES agreement_items(id),
  rating VARCHAR(20), -- 'working', 'needs_adjustment', 'not_working'
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

## Privacy & Safety

### Data Protection
- All preparation is private until explicitly shared
- Users can withdraw from session at any time
- Session data deleted after 90 days of inactivity
- No recording of conversations (text only)

### Safety Features
- Pause session if either party feels unsafe
- "This is too much right now" escape hatch
- Resource links for professional support
- Clear messaging: this is practice, not therapy

### Moderation (for future public matching)
- Shared sessions only with invited known contacts
- No stranger matching in Phase 5
- Report/block capabilities for future phases
